# ValuationIA - Base Project

Este é um scaffold mínimo para rodar localmente a base do projeto ValuationIA (Next.js + TypeScript + Tailwind).

## Passos rápidos (resumido)
1. Instalar Node.js LTS (https://nodejs.org/)
2. Abrir terminal na pasta do projeto
3. Executar `npm install`
4. Executar `npm run dev`
5. Abrir http://localhost:3000
